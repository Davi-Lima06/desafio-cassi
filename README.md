# DESAFIO CASSI

desafio proposto pela cassi

## Tecnologias Utilizadas

- Java 17+
- Spring Boot 3.3.3
- Maven

## Aplicação requesitos
- ter java 17 + instalado
- maven instalado

## Rodando a aplicação
- vá na raiz do projeto e digite: mvn spring-boot:run
- abra a url no seu navegador: http://localhost:8080
